WordPress plugin to crawl Aparat API.

== Description ==

You can use this plugin to crawl your Aparat channel and update your WordPress site.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
